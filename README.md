# What is DefaultsCloudSync?
DefaultsCloudSync is a simple Swift class that synchronizes NSUserDefaults to iCloud.

# Usage
In your AppDelegate's didFinishLaunchingWithOptions call <br> 
`DefaultsCloudSync.start()`  <br>
That's it!
